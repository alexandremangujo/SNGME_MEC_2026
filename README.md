# SNGME_MEC_2026
Repositório oficial para o projecto do Sistema Nacional de Gestão de Mobilidade e Educação a ser desenvolvido pelos estagiários.
SNGME é uma aplicação Web desenvolvida em PHP,para apoiar, automatizar e integrar os principais processos de administração de pessoal DPEC. O sistema permite o registo, controlo e acompanhamento e mapeamento das deslocações dos funcionarios afectos nas escolas, e os estágio dos processos de mobilidade.

Além disso, o sistema facilita a geração de relatórios , permitindo o acompanhamento de mobilidades e necessidades de pessoal num determinada area geografica.
